# Sales Data Dashboard | Python | SQL | Power BI

## Project Overview
End-to-end data analysis project on the Kaggle Superstore dataset 
— from raw data cleaning to an interactive 3-page Power BI dashboard.

## Tech Stack
Python | Pandas | SQL | SQLite | Power BI

## Dataset
- Source: Kaggle Superstore Dataset
- 9,994 rows | 21 columns | Line-item level sales data (2014–2017)

## Key Steps
1. Data Cleaning (Python/Pandas)
   - Removed 8 duplicate records
   - Fixed date dtypes from object to datetime64
   - Engineered new columns: Year, Month, Quarter, Ship Duration

2. SQL Analysis (SQLite)
   - Quarterly sales trends
   - Profit by order and region
   - Sub-category profitability
   - Discount vs profit breakpoint analysis

3. Power BI Dashboard (3 pages)
   - Executive Summary
   - Profitability Analysis
   - Seasonal Trends

## Key Findings
- Tables sub-category generates $207K revenue but incurs $17K loss
  due to discounts exceeding 20%
- Technology is the highest-margin category (17.4% margin)
- Furniture generates $741K revenue but only $18K profit
- Q4 is the dominant quarter every year — Technology leads peak demand
